      异步咒术

      咒术会在单个时间片段内完全施放，这是它们的本性。而为延迟咒术，自然设计了一套有些奇怪的方案：一盒磁带，需要“食用”的磁带。我最多可同时持有六盒磁带。每盒磁带都可容纳单个需延迟执行的咒术，直至可以施放时再施放。此后，该磁带会空出，供其他咒术使用。


      点击以显示配方 点击以隐藏配方
      磁带

      为促使音乐唱片与媒质融合，我为其镀上了相当多的充能紫水晶。所得的智慧结晶需要食用，魔法、意志、持之以恒地咀嚼，三者缺一不可；这一过程后，即可将其收作己用。


      入队列 (list, number → number)

      • 渲染方式
        • 配色
          浏览器不支持图案的可视化。图案代码见此：qeqwqwqwqwqeqaweqqqqqwweeweweewqdwwewewwewweweww

          令一个咒术入队列，并在所给数个 1/20 秒后施放。咒术会进入第一个空闲的磁带，或进入当前磁带，同时返回该磁带的索引。


          磁带间使用索引数区分。索引自 0 起始，至目前意识中磁带的总数之前为止。如果入队列找不到空闲的磁带，则会招致“并行处理”事故：终止所有活跃中的磁带。

          这似乎表明我所处现实的时间分辨率很低……还是别再想了。

          Ibra oniki ra. QUINIO QUINIO QUINIO QUINIO alef a ra.


          按下思索磁带键可以思索磁带，使用方向键可循环切换。已入队列的咒术会显现为插入的磁带，同时在侧面显示其标签。按压磁带可以取消对应咒术。

          死亡也会强制将所有磁带清出队列。如此就无需担心了——我不会陷入无穷无尽的死亡循环了。


          若磁带将其自身的副本稍加改动后入队列，便会出现一种特殊现象：常态、持续的施法。古代的咒术师们称之为守护进程。咒法学的这一分支并未得到深入研究，也许是因为操控时间危险性极高：磁带中咒术只会消耗物品栏中的媒质。若是不加监测，它能轻松耗光我的媒质，然后便是我的生命。


          出队列 (number →)

          • 渲染方式
            • 配色
              浏览器不支持图案的可视化。图案代码见此：eqeweweweweqedwqeeeeewwqqwqwqqweawwqwqwwqwwqwqww

              令所给索引处的磁带出队列，让其进入空闲状态，并取消其中咒术。


              终止队列

              • 渲染方式
                • 配色
                  浏览器不支持图案的可视化。图案代码见此：eqeweweweweqedwqeeeeewwqqwqwqqw

                  令所有磁带出队列。很适合给那些和衔尾蛇一样的咒术用：两个互相激活的磁带无法以常规手段出队列。

                          程序之纯化 (number → null/list of patterns)

                          • 渲染方式
                            • 配色
                              浏览器不支持图案的可视化。图案代码见此：eqeweweweweqedwqeeeee

                              若所给索引处磁带不处于活跃状态，返回 Null；否则返回其中咒术。我听过一些传言，说这个图案可以用来存储 iota，也可以用来在咒术间传递信息。


                              先知之纯化 (number → null/number)

                              • 渲染方式
                                • 配色
                                  浏览器不支持图案的可视化。图案代码见此：eqeweweweweqedwqeeeeedww

                                  若所给索引处磁带不处于活跃状态，返回 Null；否则返回该磁带会在多少个 1/20 秒后施放。


                                  入队列 (list, number, number → number)

                                  • 渲染方式
                                    • 配色
                                      浏览器不支持图案的可视化。图案代码见此：qeqwqwqwqwqeqaweqqqqqwweeweweewqdwwewewwewweweww

                                      入队列的一个变种，在咒术后还需向其提供一个索引。它会强制将该磁带中的咒术换为所给参数，无论原先里面有没有咒术。


                                      如果所给的索引不存在，则招致“内存溢出”事故：终止所有活跃中的磁带。


                                      状态之精思 (→ null/number)

                                      • 渲染方式
                                        • 配色
                                          浏览器不支持图案的可视化。图案代码见此：qeqwqwqwqwqeqaweqqqqqwweeweweew

                                          返回当前咒术所在磁带的索引。若不在磁带中运行，则返回 Null。
